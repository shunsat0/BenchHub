# こんなアプリを作っています
外に出るとベンチを探しがちな人向けアプリ。
リリースは目標は7月。
僕はよくベンチを探すので、近くのベンチを探したり、口コミを投稿できるアプリを開発中です。
ゆくゆくは、ゴミ箱や自販機など、有名なマップでは探しにくい情報を追加していく予定です。
